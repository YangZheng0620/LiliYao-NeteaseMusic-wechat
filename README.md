# 微信小程序云音乐项目

### 1. 项目介绍
LiliYao-NeteaseMusic-wechat 如果对各位朋友有用的话, 希望可以给个star⭐

#### 技术栈
- 使用微信小程序原生 WXML + WXSS + JS 进行开发
- 使用 wx.request() 进行网络请求
- CSS 使用 FLEX 进行布局
- 项目组件库使用 Vant Weapp

#### 项目使用库
- Vant Weapp
- pinyin-pro
- hy-event-store

#### 项目目标
- 使用微信小程序开发云音乐项目
